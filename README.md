# 🐋 Ballena con Canvas 2D

## 📌 Descripción
Aplicación web que dibuja una ballena usando la API Canvas 2D mediante programación, mostrando la imagen original como referencia.

## 🚀 Tecnologías
- HTML5
- JavaScript
- Canvas 2D
- Bootstrap 5

## 🎯 Objetivo
Comprender el uso de figuras básicas y trazos para generar gráficos por computadora.
